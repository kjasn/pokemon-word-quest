# Pokémon Word Quest

A fun word-guessing game where you lose a Pokémon for every wrong guess 😉!

Take a look, [click here](https://pokemon-word-quest.vercel.app/)

## Quick Start

1. Clone it.
2. Run the following commands:

```bash
pnpm i
pnpm start
```
