这是一个简单的猜单词游戏：试着猜出正确的单词，记住每猜错一次，你就会失去一只宝可梦。现在来挑战成为宝可梦大师吧！🎮✨

## 特点

-   📱 适配桌面端和移动端的响应式设计
-   🌓 支持亮暗主题切换
-   🎯 简单直观的游戏玩法
-   🎮 宝可梦主题的 UI
-   🎊 一些设计上的彩蛋（算是彩蛋吧？🙉）

## 运行项目

-   本地运行

1. 克隆到本地
2. 运行如下命令:

```bash
pnpm i
pnpm start
```

-   使用 Docker 运行

```bash
docker pull kjasn/pokemon-wq
```
