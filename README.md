## Features

-   📱 适配桌面端和移动端的响应式设计
-   🌓 支持亮暗主题切换
-   🎯 简单直观的游戏玩法
-   🎮 宝可梦主题的 UI
-   🎊 一些设计上的彩蛋（算是彩蛋吧？🙉）

## 运行项目

1. 克隆到本地
2. 运行如下命令:

```bash
pnpm i
pnpm start
```
